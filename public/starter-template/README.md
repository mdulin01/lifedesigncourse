# My Personal Website

A starter template for building your personal website with AI.

## Quick Start

1. **Install Node.js** from [nodejs.org](https://nodejs.org) (LTS version)
2. **Open your terminal** and run:

```bash
npm install
npm run dev
```

3. Open [http://localhost:5173](http://localhost:5173) in your browser
4. Open `src/App.jsx` — this is where your entire site lives
5. **Tell Claude what to change** — describe what you want in plain English

## How to Customize

You don't need to write code. Open Claude and try prompts like:

- "Replace the placeholder content with a site for me. I'm a [your role] who [what you do]."
- "Change the color scheme to navy blue and warm gold."
- "Add a Publications section with 5 of my recent papers."
- "Make the hero section full-screen with a gradient background."

## Deploy to the Internet

When you're ready to share your site:

1. Create a [GitHub](https://github.com/signup) account
2. Create a [Vercel](https://vercel.com) account (sign up with GitHub)
3. Ask Claude: "Help me push this to GitHub and deploy on Vercel"

Your site will be live at `your-project.vercel.app` in minutes.

## Tech Stack

- **React** — UI framework
- **Vite** — Build tool (fast development server)
- **Tailwind CSS** — Styling (utility classes)
- **Lucide React** — Icons (optional)
